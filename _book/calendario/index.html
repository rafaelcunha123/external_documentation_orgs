Calendário

A página de calendário permite iniciar as principais ações na operação de uma clínica. São elas:

  1. Selecionar uma unidade de atendimento
  2. Visualizar a agenda de um profissional
  3. Selecionar local de atendimento
  4. Realizar uma recepção
  5. Visualizar a sala de espera de um profissional
